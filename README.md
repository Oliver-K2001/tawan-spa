# Tawan Spa 2 – Moderne Website

Modernisiertes Website-Konzept für **Tawan Spa 2 & Thai Massage Hamburg**.

## Projektstruktur

```
tawan-spa/
├── index.html          ← Hauptseite (Landing Page)
├── massagen.html       ← Massagen & Preisübersicht
├── css/
│   └── style.css       ← Alle Styles (CSS-Variablen, Animationen)
├── js/
│   └── main.js         ← Interaktionen (Nav, Animationen, Counter)
├── assets/             ← Hier Bilder einfügen
│   └── (leer – Bilder vom Original einfügen)
└── README.md
```

## 🎨 Design-Konzept

- **Aesthetik:** Luxury / Refined Spa — dunkel, golden, elegant
- **Fonts:** Cormorant Garamond (Display) + Jost (Body) — via Google Fonts
- **Farben:** `#0C0A07` (Dunkel) · `#C9A96E` (Gold) · `#F5EDD6` (Cream)
- **Features:** Scroll-Animationen, Counter, Mobile-Menü, Sticky Nav

## 🚀 Start

Einfach `index.html` im Browser öffnen — oder mit Live Server in VS Code:

1. VS Code Extension **Live Server** installieren
2. `index.html` öffnen
3. Rechtsklick → *Open with Live Server*
